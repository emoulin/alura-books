# Alura Books

Este projeto foi uma tarefa do curso de Front-End da Alura, consiste na criação de um site de uma Livraria.

## Tecnologias Utilizadas
* HTML
* CSS

## Conteúdos abordados
* Mobile-first
* Responsividade
* Display flex e Grid
* Swiper-js
* Listas
* Formulários
